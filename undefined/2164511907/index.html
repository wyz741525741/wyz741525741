电脑是一个生产力工具，我们不能让它仅仅沦落为一个娱乐工具，新时代的少年总要跟上时代的潮流，虽然现在是45年入德军稍微有点晚，但这并不妨碍它的效率。

因为经常翻阅英文版说明书（实在是厂家不给中文的呀~）所以需要随身携带一本词典，但今天还有什么是电脑不能装的？今天就给大家推荐我自用的离线词典！不用额外带词典了！关键没网也能用！而且很全！！。

词典（GoldenDict）

简介

  • GoldenDict，一款尤其适用于计算机端（Windows、macOS、Linux）的开源、自由、可定制的词典软件。严肃的语言学习者、文字工作者乃至词典编纂者，都应该至少尝试一下 。
    • 免费开源，全平台使用。
    • 无任何广告，绿色解压就能用，实在太方便。
    • 难以置信的加载论坛上分享的mdx格式的词典，随心所欲。
    • 自带维基百科，秒开，实在太方便。
    • 可以配置真人发音，just cool!
    • 可以离线使用以及好用的屏幕取词

下载

  • 官网下载地址的版本已经很久很久没有更新了，但官方给了另一个渠道Goldendict的GitHub官方源

    • 动手能力及英文能力强👍的可以直接打开https://github.com/goldendict/goldendict
    • 动手能力及英文能力弱👀的直接点击这里然后找到下图，根据电脑的系统版本选择对应32/64位进行下载！当然我就64位走起了

    • 另外备用地址所有的历史版本这里都有（如果以上访问地址被墙的话）https://sourceforge.net/projects/goldendict/files/early%20access%20builds/

使用

  • 准备

    • 点击文件夹中的 GoldenDict.exe，你可以选择为 GoldenDict.exe 建立桌面快捷方式或者固定在任务栏，方便快速访问
    • 如图，提示“正在索引词典，请稍候”，如果词典数量较多，这个过程会比较长

  • 关键

    • 防止全文搜索疯狂占用CPU资源以及保证打开速度，我们关闭全文搜索功能（当然如果想体验请跳过此步骤）

    • 点开【编辑】->【首选项】->【全文搜索】，按照自己的需求，选择关闭全文搜索或者选择仅对某一（或几）种格式的词典进行全文搜索

    • 我按照自己的选择，选择对 DSL 、MDict 格式词典进行全文搜索。

  • 查词

  • 全文搜索

  • 查看词头

  • 词典分组

    • 【编辑】->【词典】
    • 【词典来源】选项卡里的【添加】按钮是灰色的，这是因为我们在第三步，新建了 portable

GoldenDict 帮助

按下快捷键 F1，或“菜单栏”【帮助】->【GoldenDict 帮助】，可打开 GoldenDict manual 。本文下面的内容，实际上可以看作 GoldenDict manual 的补充。

GoldenDict 支持的词典格式

支持几乎所有开源词典格式，详见 Supported Dictionary Formats，不过目前有新作的仅剩 mdx、dsl 格式词典了。

mdx/mdd

  • PDAWIKI 掌上百科 Mdict资料库
  • PDAWIKI 掌上百科编纂处 作品展示

一个精品 mdx 耗费大量时间、精力，如果你不曾亲身亲历，请尊重他人的分享方式；实在不行，你可以自己试着做一做。自己动手，丰衣足食。

dsl

  • lingvodics.com
    • 这是一个索引站点，指向 ru-board 的发布贴，需要注册帐号方可查看。

在线词典

  • GoldenDict 添加在线词典
  • 为GoldenDict添加在线有道翻译

在线词典，一般访问速度都较慢。

FAQ

  • 常见问题：官网下载、安装的 Windows 版本，不支持 mdx/mdd
    • 解决办法：去 GitHub 下载最新编译版
  • 常见问题：可下载的安装文件好几个，QT5、QT4、64bit 等是什么意思？
    • QT 是应用程序开发框架，参考百度百科，Microsoft Word 有 2003、2007……2016，QT 也有 QT4、QT5，版本号不同；
    • 如果你的操作系统是 64 位的（内存大于 4G），建议选用文件名里带 64bit 的安装包。
  • 常见问题：GoldenDict CPU 占用高
    • 解决办法：关闭 全文搜索索引。
    • 选择菜单栏【编辑】->【首选项】->【全文搜索】->【允许对以下格式词典进行全文搜索】，把相应的 √ 去掉。
    • “全文搜索”是个大杀器，建议尝试一下。
  • 常见问题：GoldenDict 安卓版 ，收费，但是国内不能在 Google Play 购买，且安卓版不支持 mdx/mdd
    • 解决办法：这个我没有相关经验，我个人主要在桌面端查词典，手机端很少用。
    • 安卓版 Goldendict 支持 .mdx .mdd 的方法 (经验)
  • 常见问题：GoldenDict 没有 iOS 版
    • 解决办法：无解，慢慢等吧。
    • GoldenDict Android (安卓系统) 官网链接下载
  • 常见问题：GoldenDict 词典发音
    • 解决办法：只需要加载含音频（或可联网发音）的 mdx/mdd 即可，不需要去设置“音频文件目录”
  • 常见问题：词典音标乱码、字体有问题
    • 解决办法：这个问题一般是使用了老旧的系统、词库，建议更新自己的操作系统，尝试使用较新的词库。
